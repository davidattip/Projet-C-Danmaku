Danmaku Ultimate est un jeu de tir à défilement vertical inspiré par le genre classique des "bullet hell" ou "Danmaku". Dans ce jeu, les joueurs doivent naviguer à travers des vagues denses de projectiles et essayer de survivre le plus longtemps possibles alors que la difficulté augmente au fil du temps. 

Fonctionnalités :

-Modes de Difficulté Variés: Choisissez entre trois niveaux de difficulté - Aspirant, Guerrier, et Berserker - pour choisir le niveau de difficultés il faut utilisé la flèche du bas puis appuyé sur entrée pour changer la difficulté et ensuite appuyer sur la flèche du haut et appuyer sur entrer pour lancer le jeu. 
-Commandes Réactives: Manœuvrez votre vaisseau avec précision grâce au flèches directionnelles.
Système de Score: Tenez le plus longtemps pour augmenter votre score qui s'affichera pendant la partie et une fois la partie terminée.

 Commandes de Jeu:

Mouvement: Utilisez les flèches directionnelles pour déplacer votre vaisseau.
Sélection dans le Menu: Utilisez les flèches directionnelles pour naviguer dans le menu et la touche Entrée pour sélectionner.

Comment Jouer:

-Lancez le jeu pour atteindre l'écran titre.
-Appuyez sur Entrée pour accéder au menu principal.
-Utilisez les flèches directionnelles pour choisir entre commencer le jeu ou modifier la difficulté.
-Sélectionnez une option avec la touche Entrée.
-Une fois en jeu, évitez les projectiles ennemis et ripostez pour obtenir le meilleur score possible.
-Le jeu se termine lorsque toutes les vies sont perdues. Vous pouvez alors redémarrer ou quitter le jeu.

Installation:

Pour jouer à Danmaku Ultimate, assurez-vous d'avoir installé SDL2, SDL2_image, SDL2_ttf, et SDL2_mixer. Compilez ensuite le code source du jeu avec un compilateur prenant en charge C.
